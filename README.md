Pneu Safe 🚗🛡️

Sistema inteligente para gerenciamento e monitoramento de pneus, desenvolvido com foco em segurança, organização e praticidade no controle de informações automotivas.

📌 Sobre o Projeto

O Pneu Safe é uma aplicação criada para auxiliar no controle e acompanhamento de pneus de veículos, permitindo uma gestão mais eficiente e segura.

O projeto foi desenvolvido com o objetivo de facilitar:

Cadastro de pneus;
Monitoramento de desgaste;
Controle de informações importantes;
Organização de dados automotivos;
Segurança e praticidade no gerenciamento.
🚀 Funcionalidades
✅ Cadastro de pneus
✅ Registro de informações do veículo
✅ Controle e monitoramento
✅ Interface moderna e intuitiva
✅ Organização de dados
✅ Sistema focado em segurança automotiva

⚙️ Como Executar o Projeto
1️⃣ Clone o repositório
git clone https://github.com/Andreisilva71/pneu-safe.git
2️⃣ Acesse a pasta
cd pneu-safe
3️⃣ Instale as dependências
npm install
4️⃣ Execute o projeto
npm run dev ou npm start
